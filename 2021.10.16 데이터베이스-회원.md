### 비밀번호 암호화
sha2('1',256) : 256비트 sha2 함수로 암호화 저장 ( 비밀번호는 1)
select 함수에서 \G를 맨 뒤에 붙이면 값을 세로로 볼 수 있음

### 비밀번호 변경
update user set password=sha2('2',256) where userId=8 and password=sha2('1',256);

### 회원 탈퇴
delete from user where userId=8 and password=sha2('1',256);
